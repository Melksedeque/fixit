# Módulo de Chamados - Visão Geral

## Objetivo

Criar um sistema de chamados moderno inspirado em:

- Trello (Kanban visual e intuitivo)
- Runrun.it (métricas, SLA e controle)
- Sistemas Helpdesk SaaS

O módulo deve permitir:

- Abertura simples de chamados
- Organização por status
- Acompanhamento por responsáveis
- Visualização em Lista e Kanban
- Métricas claras
- Histórico completo de interações

## Princípios

- UX simples para quem solicita
- Poder e controle para quem gerencia
- Visual consistente com o Design System Dark
- Zero complexidade desnecessária
- Performance e clareza

## Modos de Visualização

- Lista (padrão)
- Kanban
- Detalhe completo
