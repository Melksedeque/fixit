# Experiência do Usuário

## Abertura de Chamado

Fluxo simples:
1. Botão "Novo Chamado"
2. Modal ou página com:
   - Título
   - Descrição
   - Prioridade
   - Categoria
3. Submit
4. Feedback visual imediato

## Ao visualizar um chamado

Usuário deve:
- Ver histórico
- Adicionar comentário
- Ver status
- Ver responsável
- Ver SLA

## UX desejada

- Respostas rápidas
- Sem navegação complexa
- Interação parecida com chat
