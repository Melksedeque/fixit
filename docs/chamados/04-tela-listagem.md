# Tela de Listagem (Lista)

## Elementos

- Filtro por:
  - Status
  - Prioridade
  - Responsável
  - Busca textual

- Métricas rápidas:
  - Abertos
  - Em andamento
  - Resolvidos
  - Cancelados

- Alternador:
  [ Lista | Kanban ]

## Tabela

Colunas:

- Título
- Status
- Prioridade
- Responsável
- Criado em
- SLA
- Ações

## Interações

- Clique abre detalhe
- Hover suave
- Badge visual para status
- Badge visual para prioridade
