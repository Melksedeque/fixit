# Tela Kanban

## Estrutura

Colunas:

- Abertos
- Em andamento
- Aguardando
- Resolvidos

## Comportamento

- Drag and Drop entre colunas
- Atualização automática de status
- Contador no topo da coluna
- Scroll horizontal

## UX

- Cards compactos
- Mostram:
  - Título
  - Prioridade
  - Responsável
  - SLA
