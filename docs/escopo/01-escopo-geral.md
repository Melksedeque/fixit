# Projeto Fixit - Sistema de Chamados Multimídia

## Visão Geral

Sistema de chamados focado em comunicação rápida (áudio, vídeo, imagem) e gestão ágil de prazos.

## Tech Stack

- **Framework:** Next.js 16+ (App Router)
- **Linguagem:** TypeScript
- **Estilização:** Tailwind CSS & Lucide Icons
- **Banco de Dados:** PostgreSQL (via Neon.tech)
- **ORM:** Prisma
- **Autenticação:** NextAuth.js (Credentials Provider)
- **Validação:** Zod
- **Tratamento de Datas:** date-fns (GMT-3 / Local)
- **Segurança:** bcryptjs
- **Testes:** Jest & React Testing Library
