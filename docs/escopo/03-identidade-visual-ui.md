# Guia de Estilo & UI/UX

## Identidade

- **Nome:** Fixit
- **Fonte Principal:** Baloo Paaji (Logo), Montserrat/Inter (Interface).
- **Cores:**
  - Primary: #1B365D (Deep Blue)
  - Secondary: #48C9B0 (Teal/Green)
  - Urgência: Red-500 (Critical), Orange-500 (High), Blue-500 (Medium), Green-400 (Low).

## UX/UI Principles

- **Dashboard:** Visão Kanban (estilo Trello) e Lista (estilo Jira).
- **Componentes:** Card de chamado com timer de execução visível.
- **Input de Mensagem:** Botão de "Segure para gravar áudio" e "Upload rápido de mídia".
