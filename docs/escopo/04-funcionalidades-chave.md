# Requisitos Funcionais

1. **Gestão de Chamados:**
   - Cálculo automático de `executionTime` ao finalizar.
   - Comparativo visual entre `deadlineForecast` e `deliveryDate`.
2. **Multimídia:**
   - Integração para exibição de players de áudio e vídeo nativos no chat do ticket.
3. **Filtros:**
   - Por urgência, técnico responsável e data de entrega.
4. **Site Institucional:**
   - Landing page simples explicando o fluxo de "Reportar -> Corrigir -> Entregar".
